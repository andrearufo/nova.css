Nova.css

Welcome to nova.css home page.

Nova.css is an(other) simple and minimalistic CSS framework for your webpages. Made in pure SCSS and compiled with Gulp.

Typography

The nova.css default font is Source Sans Pro with Source Code Pro for the monospaced parts.

Code

This CSS is specialy made for code pages:

// Mixin for responsive
@mixin responsive($size) {
    @if $size == large {
        @media (min-width: $breakpoint) {
            @content;
        }
    } @else {
        @media (max-width: $breakpoint) {
            @content;
        }
    }
}
